PANDUAN RESMI
Tutorial Singkat

Ambil laporan riwayat trading
dari MetaTrader 4 / 5.

Ekspor statement (.htm/.html) dari terminal MT4/MT5 kamu dalam 4 langkah singkat. File diproses langsung di browser, tidak diunggah ke server manapun.

Langkah-Langkah Export Laporan
LANGKAH 1
Buka Tab Riwayat Akun
MT4 & MT5

Di terminal MT4/MT5 (aplikasi desktop), buka panel Toolbox di bagian bawah layar (tekan Ctrl+T kalau tersembunyi), lalu klik tab "Account History" (MT4) atau "History" (MT5).

ILUSTRASI · PANEL TOOLBOX MT4/MT5
LANGKAH 2
Pilih Rentang Waktu
MT4 & MT5

Klik kanan di area riwayat tersebut → pilih periode di menu "Custom Period" atau langsung pilih "All History" supaya seluruh riwayat trading kamu ikut ter-export.

ILUSTRASI · MENU KLIK KANAN
LANGKAH 3
Simpan sebagai Report
MT4 & MT5

Klik kanan lagi di area riwayat → pilih "Report" → "Save as Report" (MT4) atau ikon Save/printer di pojok panel (MT5). File akan tersimpan dalam format .htm.

ILUSTRASI · REPORT → SAVE AS REPORT (.HTM)
LANGKAH 4
Upload ke Thunder Trade
SELESAI

Kembali ke halaman utama Thunder Trade, lalu drag & drop file .htm tadi ke kotak upload, atau klik kotaknya untuk memilih file secara manual.

ILUSTRASI · UPLOAD KE KOTAK DROPZONE
Pertanyaan Umum
❓ FAQ
Yang Sering Ditanyakan

File-nya tidak ketemu di komputer? Cek folder Downloads atau folder tempat kamu terakhir menyimpan (default biasanya folder Desktop/Documents) — nama file umumnya seperti "ReportHistory.htm" atau "StatementReport.htm".

Pakai HP (MT4/MT5 Mobile)? Aplikasi mobile tidak bisa export file .htm langsung. Sebaiknya buka MT4/MT5 versi desktop di laptop/PC, atau login ke akun yang sama di komputer, untuk melakukan export.

Muncul opsi banyak sekali saat "Save as Report"? Pilih yang formatnya Detailed Report / Statement (bukan Excel/CSV) — Thunder Trade hanya membaca format .htm / .html.
